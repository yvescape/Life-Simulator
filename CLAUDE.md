Crée à la racine du projet un fichier CLAUDE.md qui décrit le projet Life Simulator :

- C'est un jeu de simulation de vie web où le joueur incarne un personnage
  de la naissance à la mort.
- Stack : React 18 + Vite + TypeScript + Tailwind CSS pour le frontend.
- State management : Zustand.
- Backend prévu plus tard : Node.js + Express + PostgreSQL.
- Tous les textes affichés au joueur doivent être en français.
- Le code et les commentaires sont en français.
- Les événements de jeu sont en JSON dans /src/data/events/.
- Préfère toujours des composants React fonctionnels avec hooks.

Inclus aussi une section 'Architecture' avec la structure des dossiers prévue.
